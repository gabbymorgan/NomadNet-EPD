# About NomadNet EPD

This is a fork of the robust [NomadNet](https://github.com/markqvist/NomadNet) application. This fork will remain compatible the main repo, and updates will be merged in a timely fashion for as long as it is active.

It is currently compatible with the [Waveshare 2.13in E-Paper Display](https://www.waveshare.com/2.13inch-e-Paper-HAT.htm) because that is the one I purchased.
